# bjorntheprogrammer.github.io

This is my personal website. With zero dependencies, and 6 dev dependencies. Built using `bun-plugin-html`, `tailwindcss`, and `bun-html-live-reload`.

To install dependencies:

```bash
bun install
```

To build:

```bash
bun run build
```

Hot reload dev mode:

```bash
bun run dev
```
